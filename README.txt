O PathFinder é uma ferramenta que atende a uma necessidade cada vez mais presente na sociedade, que é ter tempo e disposição para planejar um roteiro turístico completo e seguro. Com a plataforma, os usuários podem explorar o potencial turístico da cidade de São Paulo de forma mais eficiente e segura, garantindo uma experiência satisfatória e confortável.

Foi implementado um conjunto de funcionalidades no sistema, incluindo um mecanismo de Nested Dictionary que centraliza todas as informações essenciais do sistema. Essa abordagem otimiza o código, tornando-o mais conciso e legível.

O sistema agora incorpora diversas funcionalidades avançadas, como loops, retornos, parâmetros e manipulação de listas, proporcionando uma experiência mais abrangente. Ao dar início, o sistema solicita ao usuário que insira seu nome. Ele então verifica se o nome foi de fato inserido. Caso contrário, o sistema emite um alerta indicando o erro.

Após a etapa de nome, o sistema apresenta uma série de opções para os tipos de pontos turísticos que o usuário deseja visitar. Assim que a escolha é feita, o sistema direciona a atenção do usuário para selecionar um método de locomoção.

Com todas essas seleções concluídas, o sistema gera e exibe o trajeto sugerido, acompanhado da indicação da forma de transporte escolhida. Consequentemente, ele pergunta se o usuário deseja continuar utilizando o programa. Caso o usuário opte por continuar ("SIM"), o sistema é reiniciado e está pronto para ser usado novamente. Por outro lado, se a escolha for interromper o uso ("NÃO"), o sistema é encerrado com uma despedida apropriada.

Destaca-se que esta versão do sistema incorpora um sistema de tratamento de erros robusto. O usuário é notificado de quaisquer erros que possam ocorrer ao longo do processo, garantindo um feedback claro e útil para orientar suas interações.

Por meio dessas melhorias, o sistema oferece uma experiência mais fluida, interativa e eficiente, proporcionando aos usuários uma maneira conveniente e confiável de planejar suas rotas de preferência.
